# red-desktop-hello-world

## Description
A POC for red programming language.

## Tech stack
- red

## Docker stack
- ubuntu:22.04
    - i386 architecture

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
